## 説明文（4000文字以内 / 実測: 約1100文字）

```
中1テストキットは、中学1年生の中間テスト・期末テストに特化した5教科対策アプリです。
テスト日までのカウントダウン、単元ごとの解説、ドリル、予想模試を一つにまとめました。
教科書を開く時間がない日も、スマホで5分の積み重ねで点数につながります。

▼ こんな中学1年生におすすめ
・はじめての定期テストで何をやればいいかわからない
・部活や習い事で勉強時間がまとまって取れない
・ワークは解いたけど、テストでどう出るかイメージが湧かない
・苦手な単元だけ集中して復習したい

▼ 中1テストキットの特徴

1. テストカウントダウンで集中力を保つ
ホーム画面に「あと◯日」と表示。テスト日程を入れるだけで、毎日の学習にちょうどいい緊張感が生まれます。

2. 5教科 × 全25単元の解説とドリル
英語・数学・国語・理科・社会の主要単元を、教科書の流れに沿って収録。各単元は「解説 → ドリル」の順で進めるので、迷わず学習できます。

3. 予想模試で本番感覚を養う
中間モード（主要8単元）と期末モード（全25単元）を切り替え可能。5教科からランダムに各5問、合計25問の模擬テストで、本番直前のチェックに最適です。

4. わからない所はAIに質問
単元ごとに「AIに質問する」ボタン。教科書の例文や問題でつまずいた所を、自分の言葉で聞けます。1日30回まで質問可能。

5. 教科書出版社に対応
東京書籍・光村図書・三省堂など、主要な教科書出版社別に内容を調整できます。

▼ 料金について
基本機能は無料、すべてのコンテンツへのアクセスは¥980の買い切り（一度の支払いで永続利用可能）。月額・年額サブスクリプションはありません。家計にも負担をかけません。

▼ 中学1年生・保護者の方へ
中1の1学期は、これまでの小学校とは違う「定期テスト」というハードルが初めて現れる時期です。本アプリは「最初のテストで自信をつける」ことを目標に、最短ルートで点数につながる単元と問題に絞って設計されています。

▼ 制作者より
塾講師・教材設計の経験をもとに、中学1年生が本当につまずくポイントを徹底的に洗い出して作りました。フィードバックは設定画面の「お問い合わせ」からいつでも歓迎します。

定期テストを「やればできる」に変える一歩を、今日から。
```
